**使用**

后续合并后，可以 Obsidian 的官方主题仓库直接搜索 `Virgo` 进行安装。

目前需要手动操作，如下：

```
# 进行你的 Obsidian 仓库根目录中的 .obsidian
git clone https://github.com/loveminimal/obsidian-theme-virgo.git themes/Virgo
```

之后在 Obsidian 设置中选择该主题即可。
